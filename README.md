# Snake3d
Modern C++ Opengl 3D Snake example.

### Controls:
- J, I, K, L - snake move
- R - hide / show radar
- M - stop / play music
- V - show / hide shadows
- B - enable / disable blur effect
- F - enable / disable fog effect
- W - enable / disable rain and rain drops
- 1 - switch to cube head
- 2 - switch to pacman head
